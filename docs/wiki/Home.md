# E-ink Studio — Wiki

A **visual editor for ESPHome e-paper displays**, running as a Home Assistant add-on.
Drag elements onto a paper-accurate canvas, bind them to live Home Assistant values, and
generate ready-to-paste ESPHome `display:` lambda + YAML.
