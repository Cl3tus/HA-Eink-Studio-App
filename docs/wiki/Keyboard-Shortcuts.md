# Keyboard Shortcuts

## 🇬🇧 English

| Key | Action |
|-----|--------|
| **Ctrl+Z** / **Ctrl+Y** | Undo / Redo |
| **Ctrl+D** | Duplicate |
| **Ctrl+C** / **Ctrl+X** / **Ctrl+V** | Copy / Cut / Paste (paste keeps position, even on the other screen) |
| **Ctrl+A** | Select all (or select the YAML when the code drawer is open) |
| **Del** / **Backspace** | Delete selection |
| **Arrow keys** | Nudge 1 px (**Shift** = 10 px) |
| **Shift** while dragging | Bypass grid/ruler snap |
| **Ctrl/Meta + click** | Add/remove an element from the selection |
| **Shift + click** (Layers) | Range-select in the layers list |
| **Right-click** (canvas) | Context menu (layer order, delete, …) |
| **Right-click** (ruler marker) | Remove that guide |

## 🇳🇱 Nederlands

| Toets | Actie |
|-------|-------|
| **Ctrl+Z** / **Ctrl+Y** | Ongedaan maken / Opnieuw |
| **Ctrl+D** | Dupliceren |
| **Ctrl+C** / **Ctrl+X** / **Ctrl+V** | Kopiëren / Knippen / Plakken (plakken behoudt positie, ook op het andere scherm) |
| **Ctrl+A** | Alles selecteren (of de YAML selecteren als de code-lade open is) |
| **Del** / **Backspace** | Selectie verwijderen |
| **Pijltoetsen** | 1 px verschuiven (**Shift** = 10 px) |
| **Shift** tijdens slepen | Snap (raster/lineaal) overslaan |
| **Ctrl/Meta + klik** | Element toevoegen/verwijderen uit selectie |
| **Shift + klik** (Lagen) | Bereik selecteren in de lagenlijst |
| **Rechtsklik** (canvas) | Contextmenu (laagvolgorde, verwijderen, …) |
| **Rechtsklik** (liniaal-marker) | Die hulplijn verwijderen |
