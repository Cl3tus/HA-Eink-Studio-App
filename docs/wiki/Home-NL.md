# E-ink Studio — Wiki (Nederlands)

*🇬🇧 [English](Home-EN) · 🌐 [Taalkeuze](Home)*

Een **visuele editor voor ESPHome e-paper-displays**, als Home Assistant-add-on. Sleep
elementen op een papier-nauwkeurig canvas, koppel ze aan live Home Assistant-waarden en
genereer kant-en-klare ESPHome `display:`-lambda + YAML. Het is **alleen-preview** — het
schrijft nooit naar je device-config of je HA-states.
