### Capture tips

- Use a realistic profile (e.g. the 7.5" panel) so proportions look right.
- Dark mode is the default look; a light-mode variant is nice but not required.
- Hide any personal entity names if you'd rather not show them.
- Export at the window's native resolution (no upscaling); PNG.
- After adding the PNGs, they appear automatically in the README/DOCS/Wiki — no path
  edits needed (names already match).
